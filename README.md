# 📖 Intania's Lab: Archive

[![Database Supabase](https://img.shields.io/badge/database-supabase-blue?logo=supabase)](https://supabase.com)
[![Role: Engineering Student](https://img.shields.io/badge/focus-engineering_students-red)](#)

A high-performance, minimalist academic ecosystem designed specifically for engineering students. **Intania's Lab** serves as a centralized hub to manage the heavy technical workload of an engineering curriculum with a refined, architectural UI.

---

## Deployment & Backend

- **Deployment**: Hosted on **Vercel** for lightning-fast global delivery and seamless CI/CD.
- **Database**: Powered by **Supabase** (PostgreSQL) for secure, real-time data persistence.
- **Security**: Implements individual user authentication ensuring that every student's data is isolated and private.

---

## Key Functions

### Smart Calendar & Task Manager
A dual-view organizational tool tailored for academic deadlines.
* **Contextual Logic**: Automatically hides the calendar grid when viewing tasks to provide a clean, focused **Drop-down List View**.
* **User-Specific Archive**: Personal data isolation—your tasks and events are yours alone.
* **Minimalist Detail UI**: Sleek modal headers with subtle color-coded indicators (Green for Tasks, Blue for Events).

### Course Summary
* **Academic Hub**: A streamlined repository for engineering course materials and summaries, allowing for quick revision and subject tracking.

### Graph Plotter
* **Data Visualization**: Built-in tool for plotting mathematical functions and engineering data sets with a clutter-free interface.

### Equation Searcher
* **Collaborative Integration**: A specialized search engine for engineering formulas, created in collaboration with a partner project. 
* **Fast Retrieval**: Instantly find specific equations for Physics, Calculus, and specialized Engineering modules.

### Grade Predictor
* **Outcome Simulation**: Calculate and predict GPA outcomes based on current performance and target grades.
* **Priority Management**: Helps students identify which subjects require more focus to reach their target GPA.

## Design Philosophy
The **Intania Archive** follows a **"Zen-Technical"** aesthetic:
* **High Scannability**: Large margins and tight typography for an editorial feel.
* **Intentional Color**: We avoid decorative colors. Accents are strictly functional, used only to distinguish between data types (e.g., Task vs. Event).
* **Reduced Cognitive Load**: Soft backgrounds and thin borders to keep the focus on the data, not the UI.
